# Single Page CV

This project is a simple single-page CV built with HTML.

## Run the project

1. Open the file named `single page cv.html` in your browser.
2. If you prefer, you can also use a simple local server, for example:
   - Open the project folder in VS Code.
   - Right-click the HTML file and choose "Open with Live Server" if you have the extension installed.

## Files

- `single page cv.html` - Main CV page

## Notes

No additional dependencies are required.
